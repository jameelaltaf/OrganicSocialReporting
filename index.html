Drop your file here or click to browse
Supports CSV and Excel (.xlsx, .xls) exports from any platform
Where to export your data:
📸 Instagram / Facebook → Meta Business Suite → Insights → Export Data
🎵 TikTok → TikTok Studio → Analytics → Export
▶️ YouTube → YouTube Studio → Analytics → Advanced Mode → Export current view

Paste the URL of a Google Sheet containing your analytics data.
The sheet must be shared as "Anyone with the link → Viewer".

⚙️ API Configuration ▼
Get a free key at console.cloud.google.com → New Project → Enable Google Sheets API → Credentials → API Key.
Add your site domain under API key restrictions.

Sign in with Google to pull your YouTube analytics directly for the selected reporting month.
Fetches: subscribers, views, watch time, likes, Impressions CTR, avg. view duration.

⚙️ Google API Configuration ▼
Setup at console.cloud.google.com:
1. Create a project · Enable YouTube Data API v3 and YouTube Analytics API v2
2. Credentials → OAuth 2.0 Client ID → Web app → add your site URL as Authorized JavaScript Origin
3. Credentials → API Key (also needed for Analytics queries)
